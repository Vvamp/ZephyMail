## Naming Conventions

### Classes
- **Format**: `PascalCase`

### Methods
- **Format**: `PascalCase`

### Member Variables
- **Format**: `camelCaseUnderscored_`

### Local Variables
- **Format**: `camelCase`

### Parameters
- **Format**: `camelCase`

### Constants
- **Format**: `UPPER_CASE`

### Namespaces
- **Format**: `lowercase`

### Macros
- **Format**: `UPPER_CASE`

### Templates
- **Format**: Single upper-case letter (`T`, `U`, `P`, etc.)

---

## File Naming

### Header Files
- **Extension**: `.hpp`

### Source Files
- **Extension**: `.cpp`

### Class Files
- **Naming**: `PascalCase`

### Other Files
- **Naming**: `PascalCase`
